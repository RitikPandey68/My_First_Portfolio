## ⚙️ Running Locally

### Option 1: Static HTML (No server needed)
Just open `index.html` directly in your browser — all pages work as static HTML.

### Option 2: Python Flask backend (for contact form)
```bash
# Clone the repository
git clone https://github.com/RitikPandey68/My_First_Portfolio.git
cd My_First_Portfolio

# Install dependencies
pip install -r requirements.txt

# Run Flask server
python app.py
```
Then open `http://localhost:5000` in your browser.
